## Deployment

To serve the site locally:

1. **Install [Hugo](https://gohugo.io/getting-started/installing/)** (extended version recommended).

   To verify that you have the extended version installed, run:

   ```bash
   hugo version
   ```

   Ensure the output contains `extended`.

2. Clone this repository:

   ```bash
   git clone --recursive https://github.com/ki-seki/ki-seki.github.io.git
   cd ki-seki.github.io
   ```

3. If you have already cloned the repository, you can initialize the submodules with:

   ```bash
   git submodule update --init --recursive
   ```

4. Start the local server:

   ```bash
   hugo server
   ```

   The site will be available at http://localhost:1313.
