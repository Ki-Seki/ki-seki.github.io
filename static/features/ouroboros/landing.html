Ouroboros

A self-editing HTML application powered by AI

由 AI 驱动的自我编辑 HTML 应用

The snake that eats its own tail — a web app that rewrites itself.

衔尾蛇——一个能重写自身的网页应用。

What can it do?

它能做什么？

🔄

Self-modification

自我修改

Redesign its own layout, logic, or style — even delete itself.

重新设计自身的布局、逻辑或样式，甚至删除自身。

🎨

Rapid prototyping

快速原型设计

Design and iterate on UI prototypes like Google Stitch.

像 Google Stitch 一样快速设计和迭代 UI 原型。

🎮

Mini-games

小游戏

Build Snake, Tetris, or any browser game on the fly.

即时构建贪吃蛇、俄罗斯方块或任意浏览器小游戏。

🤖

Self-evolving agent

自进化 Agent

Turn the page into a multi-step autonomous workflow agent.

将页面变成能执行多步操作的自主工作流 agent。

🖥️

Simulated OS

模拟操作系统

Evolve it into a Windows-like desktop environment.

让它进化成类 Windows 的桌面环境。

📦

Single-file distribution

单文件分发

Export any evolved app as a standalone HTML file to share.

将进化后的应用导出为独立 HTML 文件进行分享。

The upper limit of its capabilities depends on the underlying model. Currently Gemini 3 Flash Preview offers the best speed–quality balance.

能力上限取决于基模水平。综合来看，目前 Gemini 3 Flash Preview 是最佳选择。

Getting started

快速上手

  1. 1

    Enter your API Key, Base URL, and Model Name in the settings panel.

    在设置面板中填写您的 API Key、Base URL 和 Model Name。

    Any OpenAI-compatible API endpoint works — OpenRouter, OpenAI, etc.

    任何 OpenAI 兼容的 API 均可使用，如 OpenRouter、OpenAI 等。

  2. 2

    Click Save Settings to store your configuration in the browser.

    点击 Save Settings，将配置保存在浏览器本地。

  3. 3

    Type your request or idea in the command input box.

    在 指令输入框 中输入您的需求或想法。

  4. 4

    Click Take a Bite 🐍 or press Ctrl+Enter to run.

    点击 Take a Bite 🐍 或按 Ctrl+Enter 执行。

Your settings are stored locally and never sent anywhere.

您的配置仅保存在本地，不会上传至任何服务器。
