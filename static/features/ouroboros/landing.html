Quick Start

    Setup API

    基础设置

    在设置内填写 API Key、Base URL 及模型名称。（兼容 OpenRouter）

  2. 2

    Save Config

    保存配置

    Click Save Settings to lock it in your cache.

    点击 Save Settings 保存到浏览器缓存中。

  3. 3

    Prompt

    输入指令

    Type what you want the app to become using natural language.

    用大白话输入你想要生成什么样的新页面或功能。

  4. 4

    Execute

    运行代码

    Hit Take a Bite 🐍 or Ctrl+Enter.

    按下 Take a Bite 🐍 或按 Ctrl+Enter。

CORS Notice

CORS 提示

Some API providers block browser cross-origin requests (CORS), so direct calls may fail even with a valid API key. We recommend OpenRouter first because browser-side CORS support is generally more reliable.

部分 API 提供商会拦截浏览器跨域请求（CORS），即使 API Key 正确也可能无法直连。建议优先使用 OpenRouter，因为它对浏览器端 CORS 的支持通常更稳定。

Security & Privacy Notice

安全与隐私提示

Ouroboros executes AI-generated JavaScript in your browser and stores API settings in localStorage. Use a dedicated API key, avoid entering sensitive personal or confidential data, and review generated behavior carefully before sharing exported files.

Ouroboros 会在浏览器中执行 AI 生成的 JavaScript，并将 API 配置存储在 localStorage。请使用专用 API Key，避免输入个人隐私或机密信息，并在分享导出的文件前仔细检查其行为与内容。
